# Resume_Ranking

## Overview
This is a Flutter mobile application developed as a final project for Mobile Programming. It showcases a user profile system with real-time data integration using Supabase. The app features a clean "Matcha" inspired UI theme and comprehensive profile management capabilities.

## Features
- **Profile Management**: View and edit user profile details including name, bio, and avatar.
- **Edit Profile**: Form with validation to update user information.
- **Friends List**: View a list of friends (integrated with Supabase).
- **Settings/About**: Application settings and information.
- **Dynamic UI**: Responsive design with a custom "Matcha" color palette.
- **Real-time Data**: powered by Supabase for data persistence.

## Tech Stack
- **Framework**: Flutter (Dart)
- **Backend**: Supabase (PostgreSQL, Auth)
- **State Management**: Provider
- **Platform**: Android

## Getting Started

### Prerequisites
- Flutter SDK
- Android Studio / Android SDK

### Installation
1. Clone the repository
2. Run `flutter pub get`
3. Configure `.env` with your Supabase credentials
4. Run `flutter run`
